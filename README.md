# My WorkShop_Muc

a simple RAG demo

### Prerequisites

- [uv](https://docs.astral.sh/uv/) installed on your system.
- example: pip install uv 

### Installation
uv sync
uv run main.py

### Further ToDos
Copy your NVIDIA_API Key in ".env" after OPENAI_API_KEY= without ":
OPENAI_API_KEY=nvapi-yourNvidiaApiKey
